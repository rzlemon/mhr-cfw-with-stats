## آموزش مشاهده آمار مصرف برای پروژه mhr-cfw 📊

---

### فایل‌هایی که دریافت می‌کنید:

| فایل | محل قرارگیری |
|------|---------------|
| request_counter.py | پوشه اصلی پروژه (کنار main.py) |
| live_status.bat | پوشه اصلی پروژه |
| proxy_server.py | پوشه src/ (جایگزین فایل قبلی) |

---

### مراحل نصب:

۱. فایل‌ها را کپی کنید
- request_counter.py و live_status.bat را در فولدر اصلی بریزید
- proxy_server.py را در پوشه src/ جایگزین کنید

۲. برنامه اصلی  را اجرا کنید
run.bat
یا از طریق
python main.py


۳. برای دیدن آمار
- فایل live_status.bat را دوبار کلیک کنید
